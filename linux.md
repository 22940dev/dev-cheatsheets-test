# Ubuntu

## Get build info

```sh
uname -a
# Linux DESKTOP-KCGTGRV 4.4.0-43-Microsoft #1-Microsoft Wed Dec 31 14:42:53 PST 2014 x86_64 x86_64 x86_64 GNU/Linux
```

## Get kernel version

```sh
uname -r
```

## Get architecture

```sh
dpkg --print-architecture
# amd64
```

## See what Linux version

```sh
cat /etc/issue
# Debian GNU/Linux 9 \n \l
```
