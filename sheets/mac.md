## Screen snip

Select an area to save to a file

```bash
cmd + shift + 4
```

Screen snip to clipboard

```bash
cmd + shift + ctrl + 4
```

## Screenshot

Window screenshot

```bash
cmd + shift + 4 #then spacebar
```

Window screenshot to clipboard

```bash
cmd + shift + cmd + 4 #then spacebar
```

## Keys

# character

```bash
alt + 3
```

## System

Show hidden files

```bash
shift + cmd + .
```
