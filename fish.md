# Fish shell


## Aliases

To list what aliases you have already defined:

```shell
ls -la ~/.config/fish/functions/
```

To see what is contained in the functions, use:

```shell
type functionName
```

To create an alias:

```shell
alias rm=trash
```

To save said alias:

```shell
funcsave rm
```

To remove an alias:

```shell
rm ~/.config/fish/functions/aliasname.fish
```
